**日期：** 2026-04-23

## 1. 背景与目标

### 1.1 功能定位

一个 Learning Agent，以"友善教练"角色，通过多轮对话引导员工学习课件内容，并在学习过程中智能评估员工掌握程度。

### 1.2 核心能力

- **主动导览**：AI 分段引导学习，确认员工理解后推进
- **智能问答**：员工可随时提问，AI 结合课件内容解答
- **学习测验**：AI 通过开放式问答检验员工掌握程度

### 1.3 交互风格

友善教练风格：鼓励式引导，穿插案例和实践建议，轻松氛围

### 2.2 组件职责

| 组件 | 职责 |
|------|------|
| TourEngine | 将课件分段，主动生成导览问题，引导员工跟读 |
| QAEngine | 理解员工问题，结合课件内容生成回答，支持多轮上下文 |
| QuizEngine | 根据学习内容智能生成开放式测验题，评估理解程度 |
| LearningFSM | 管理学习状态：导览→问答→测验→总结，控制流程推进 |

## 3. 对话流程

### 3.1 完整流程

```
┌──────────────────────────────────────────────┐
│ 1. 导览阶段 (Tour)                            │
│    AI: "欢迎学习《XXX》，我将带你了解..."        │
│    AI: "首先，我们来看第一章..."                │
│    AI: "你能用自己的话总结一下吗？"             │
└─────────────────────┬────────────────────────┘
                      │ 员工理解确认
                      ▼
┌──────────────────────────────────────────────┐
│ 2. 问答阶段 (Q&A)                            │
│    员工: "这个流程具体怎么操作？"              │
│    AI: "好问题！让我详细解释..."              │
│    AI: "除了刚才说的，还有..."                │
└─────────────────────┬────────────────────────┘
                      │ 员工提问
                      ▼
┌──────────────────────────────────────────────┐
│ 3. 测验阶段 (Quiz)                            │
│    AI: "现在我们来检验一下学习效果..."         │
│    AI: "如果XXX情况下，你会怎么处理？请说明"    │
│    AI: "回答得很好！最后一个问题..."            │
└─────────────────────┬────────────────────────┘
                      │ 完成测验
                      ▼
┌──────────────────────────────────────────────┐
│ 4. 总结阶段                                  │
│    AI: "恭喜完成学习！核心要点回顾..."          │
│    AI: "建议：1... 2... 3..."                │
└──────────────────────────────────────────────┘
```

### 3.2 状态机定义

```
tour
    │ AI 引导完成 + 员工理解
    ▼
q_and_a
    │ 员工主动说"开始测验"或 AI 检测已掌握
    ▼
quiz
    │ 完成测验（3-5道）
    ▼
summary
    │ AI 总结完成
    ▼
completed
```

## 5. 技术实现

### 5.1 API 设计

#### SSE 流式对话接口
```
POST /api/training/learn/{task_id}/chat
Request: { "message": "员工输入", "context": {...} }
Response: SSE stream of AI responses
```

#### 消息格式
```json
{
  "type": "message|quiz|summary|tour_end|q_and_a_end",
  "content": "AI 回复内容",
  "quiz_item": { ... },  // 仅 type=quiz 时
  "metadata": { "section": "第一章", "stage": "tour" }
}
```

### 5.2 Learning Agent 实现

位置：`gbrain/plugins/training/learning_agent.py`

```python
class LearningAgent:
    def __init__(self, task_id: str, content: str):
        self.task_id = task_id
        self.content = content
        self.stage = "tour"  # tour -> q_and_a -> quiz -> summary -> completed
        self.context = []  # 对话历史

    async def chat(self, message: str) -> AsyncIterator[dict]:
        """处理员工消息，返回 AI 回复流"""
        if self.stage == "tour":
            yield from self._tour_phase(message)
        elif self.stage == "q_and_a":
            yield from self._q_and_a_phase(message)
        elif self.stage == "quiz":
            yield from self._quiz_phase(message)
        elif self.stage == "summary":
            yield from self._summary_phase(message)
```

### 5.3 LLM 调用

复用现有 `course_gen.py` 中的 `call_llm` 函数，根据阶段调用不同系统提示词。

## 8. 成功标准

| 指标 | 目标 |
|------|------|
| AI 导览覆盖率 | 每个章节至少 1 次主动导览 |

| 测验完成率 | 进入测验的员工 >= 80% |
| 界面响应速度 | AI 首字延迟 < 2s |
