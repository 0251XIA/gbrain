# 场景生成器 - 基于规则的结构化提取

## 一、知识点提取规则

### 1.1 知识点来源优先级

| 优先级 | 来源 | 示例 | 提取方式 |
|--------|------|------|----------|
| P1 | ### 三级标题 | `### 1. 握手礼仪` | 直接取标题内容 |
| P2 | 有序号的重要行 | `1. 称呼礼仪 2. 着装规范` | 正则匹配数字序号 |
| P3 | 加粗/高亮关键词 | `**差旅费报销流程**` | markdown 加粗提取 |
| P4 | 表格标题 | `\| 项目 \| 要求 \|` | 提取第一列作为知识点 |
| P5 | ## 二级标题 | `## 差旅费用管理` | 取标题作为大类知识点 |

### 1.2 知识点提取逻辑

```
讲义内容
  ↓
Step 1: 按 ## 分割成模块
  ↓
Step 2: 每个模块内按 ### 分割成小节
  ↓
Step 3: 每个小节提取知识点
        - P1: ### 标题 → 直接作为知识点
        - P2: 有序号的列表行 → 拆分成多个知识点
        - P3: 加粗文本 → 作为关键知识点
        - P4: 表格 → 第一列作为知识点
        - P5: ### 标题 → 作为默认知识点
  ↓
Step 4: 合并去重，按重要性排序
  ↓
Step 5: 根据知识点数量决定场景数量
        - 最少 3 个场景
        - 最多 8 个场景
        - 每 2-3 个知识点对应 1 个场景
  ↓
Step 6: 每个知识点生成一个场景
```

### 1.3 知识点数量 → 场景数量映射

```python
# 核心公式：每个知识点 1-2 个场景
num_kp = len(knowledge_points)
min_scenes = num_kp                    # 每个知识点至少 1 个场景
max_scenes = min(num_kp * 2, 8)       # 每个知识点最多 2 个场景，上限 8
num_scenes = max(3, min_scenes)        # 最少 3 个场景
```

### 1.4 场景数量限制

| 参数 | 值 | 说明 |
|------|-----|------|
| 最少场景数 | 3 | 保证基本学习体验 |
| 最多场景数 | 8 | 避免学习时间过长 |
| 每知识点场景 | 1-2 个 | 每个知识点分配 1-2 个场景 |

## 三、知识点 → 场景 映射

### 3.1 映射规则

```
知识点列表: [KP1, KP2, KP3, KP4]
场景数: 4-8（每个知识点 1-2 个场景）
分配:
  场景1-2 → KP1 (差旅费报销流程)
  场景3-4 → KP2 (发票粘贴规范)
  场景5-6 → KP3 (报销时限要求)
  场景7-8 → KP4 (特殊费用处理)
```

### 3.2 场景数量计算

```python
num_kp = len(knowledge_points)
min_scenes = num_kp                    # 每个知识点至少 1 个场景
max_scenes = min(num_kp * 2, 8)        # 每个知识点最多 2 个场景，上限 8
num_scenes = max(3, min_scenes)         # 最少 3 个场景
```

### 3.3 映射示例

| 知识点数 | 最小场景 | 最大场景 | 实际场景 |
|---------|---------|---------|---------|
| 2 | 2 | 4 | 3 (最少3个) |
| 3 | 3 | 6 | 3 |
| 4 | 4 | 8 | 4 |
| 5 | 5 | 8 (cap) | 5 |
| 6 | 6 | 8 (cap) | 6 |
| 7 | 7 | 8 (cap) | 7 |
| 8+ | 8 (cap) | 8 (cap) | 8 |

每个场景包含：
- `title`: 从知识点生成，如 "场景1：握手礼仪"
- `description`: 基于知识点构建场景描述
- `knowledge_points`: 关联的知识点列表
- `correct_answer`: 基于知识点生成标准答案
- `hint`: 简短的引导提示

## 四、优先级提取示例

### 示例输入

```markdown
## 商务礼仪

### 1. 称呼礼仪
在商务场合，正确称呼对方非常重要...

### 2. 着装规范
商务着装要求：**整洁大方**，男士西装领带...

### 3. 名片交换
名片交换注意事项：
1. 双手递接名片
2. 名片正面朝上
3. 阅读对方姓名职位

## 财务管理

### 差旅费报销
差旅费报销流程：
| 步骤 | 操作 |
|------|------|
| 1 | 填写报销单 |
| 2 | 粘贴发票 |
```

### 知识点提取结果

```
P1 (### 标题):
["称呼礼仪", "着装规范", "名片交换", "差旅费报销"]

P2 (序号列表):
["双手递接名片", "名片正面朝上", "阅读对方姓名职位"]

P3 (加粗):
["整洁大方"]

P4 (表格):
["填写报销单", "粘贴发票"]

合并去重后共 8 个知识点 → 每个知识点 1-2 个场景 → 8 个场景
```
