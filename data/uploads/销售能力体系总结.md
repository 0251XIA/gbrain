# 契约锁销售能力体系总结

> 基于契约锁销售能力文档整理

## 一、销售闭环六阶段

| 阶段 | 核心任务 | 关键动作 |
|------|----------|----------|
| 1. 基础了解 | 精准定位客户 | 线索挖掘→判断有效性→首次拜访 |
| 2. 需求价值 | 传递产品价值 | 痛点挖掘→产品介绍→方案呈现 |
| 3. 演示阶段 | 建立信任 | 三段论（挖坑→埋土→种花）、差异化 |
| 4. 认证试用 | 推动体验 | 指导配置、行业最佳实践 |
| 5. 商务谈判 | 达成合作 | 三档报价、赠券策略、底线不破 |
| 6. 服务上线 | 交付价值 | 启动会→配置→培训→验收 |

## 二、销售四力

| 能力 | 说明 | 提升方式 |
|------|------|----------|
| 挖掘力 | 发现客户需求的能力 | 产品学习、案例学习、行业知识 |
| 沟通力 | 传递价值、建立信任 | 顾问式提问、场景化讲解 |
| 谈判力 | 商务博弈、促成成交 | 价格策略、赠券技巧、以退为进 |
| 服务力 | 持续经营、续费增长 | 客户关怀、需求响应、价值挖掘 |

## 三、产品学习四步法

1. **定位功能** — 清晰每个功能点位置
2. **理解作用** — 知道能解决什么问题
3. **掌握逻辑** — 理解背后业务逻辑
4. **串联功能** — 由点到线到面，构建体系

**升华：** 基于功能逻辑提出产品建议，展现专业性

## 四、客户画像要素

| 判断因素 | 说明 |
|----------|------|
| 需求场景 | 数字身份/电子签章/印章管控/数据存证 |
| 使用量 | 几份/百十份/千把份/万把份 |
| 预算 | 是否有预算，预算是否匹配 |
| 使用时间 | 打算什么时候使用 |
| 决策链 | 是否是决策人，决策流程是什么 |

## 五、谈判技巧

| 技巧 | 应用场景 |
|------|----------|
| 甩锅法 | 价格太低，商务不批 |
| 降价不过两 | 不能一直让价，留空间 |
| 赠券用起来 | 单价不降，代金券变相降价 |
| 三档报价 | 主推第三档，绿叶配红花 |
| 以退为进 | 站在客户角度给建议 |
| 提前堵嘴 | 老客户原价续约 |

## 六、续费沟通要点

- **服务自评**：关注使用情况、及时关怀、保持粘度
- **谈判底线**：单价不轻易降，赠券灵活运用
- **问题处理**：不惯着、不欺骗、真话不全说

## 七、学习体系

| 方向 | 内容 | 入口 |
|------|------|------|
| 产品知识 | 功能、优劣势、逻辑 | 使用教程、产品亮点、协作区 |
| 具体应用 | 客户场景、行业方案 | 公众号、市场资料、案例清单 |
| 行业合作 | 合作伙伴、系统集成 | ERP官网、集成案例 |

## 核心心态

> **1/3 是我们必须拿下来的，1/3 是我们努力争取拿下来的，还有 1/3 是我们拿不下来的。**

> **知彼知己才能扬长避短** — 产品要专业、行业要了解、友商要熟悉
