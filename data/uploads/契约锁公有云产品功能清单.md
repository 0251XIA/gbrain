## 一、签署能力

### 1.1 发起签署

| 功能 | 说明 |
|------|------|
| 本地文件上传 | 支持上传本地文件发起签署 |
| 模板发起 | 选择已上传的模板发起签署 |
| 批量发起 | 通过Excel表格批量导入发起 |
| 扫码发起 | 生成二维码/链接，无需明确接收方即可发起 |
| 继续发送签署 | 发起后可追加签署人 |

### 1.2 签署方式

| 功能 | 说明 |
|------|------|
| 签署页面签署 | 获取签署链接进行在线签署 |
| 扫码签署 | 接收方扫码签署 |
| 批量签署 | 批量盖章/签字 |
| 一键签署 | 预设位置一键完成签署 |
| 自动盖章 | 设置自动签署自动完成盖章 |

### 1.3 签署方设置

| 功能 | 说明 |
|------|------|
| 个人签署方 | 支持个人用户签署 |
| 企业签署方 | 支持企业用户签署 |
| 多方签署 | 支持2方及以上签署 |
| 顺序签署 | 按顺序依次签署 |
| 无序签署 | 多方可同时签署 |

## 十五、产品亮点总结

1. **功能全面**：覆盖签署全流程，功能丰富
2. **安全合规**：数字证书+区块链存证，法律效力强
3. **灵活便捷**：支持多种签署方式，手机端全覆盖
4. **管理规范**：三级权限体系，责权分明
5. **对接能力强**：API/SDK完善，支持多系统对接
6. **服务完善**：7*24小时客服支持
